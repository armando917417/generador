Bienvenido

La finalidad de un generador eólico casero es aprovechar la energía del viento para transformarla en electricidad de manera sencilla y económica. En este caso, utilizaremos la energía generada para cargar una batería de motocicleta, lo que nos permitirá almacenarla y usarla después para distintos propósitos. Este tipo de proyecto es útil para aprender sobre energías renovables y fomentar el uso de alternativas más sostenibles.
